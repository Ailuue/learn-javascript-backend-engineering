# Practice-Backends

A repository for learning Backend Engineering with JavaScript.

## Contents

- [d-structs-algos/](d-structs-algos/) — data structures and algorithms
- [backends/](backends/) — Express + Prisma backend projects (bookmark manager, URL shortener)

## Getting started

```bash
npm install
npm run prisma:generate   # generate the per-project Prisma clients
npm test                  # run the full Jest suite
```
